步骤详情

第一步：安装软件（电脑端）

Mac（首先需要安装Homebrew）

在命令行中，敲

brew cask install android-platform-tools

安装 ADB(Android debug bridge)，中文名称：安卓调试桥

Windows

到官网：https://developer.android.com/studio/releases/platform-tools 下载

如果有问题（由于众所周知的未知原因），请看这篇博客中的介绍：

http://xb1946.blogspot.com/2019/10/adbandroid.html

第二步：开启开发者模式（手机端）

  1. 进入设置
  2. 打开关于手机
  3. 双击“版本号”所在的行
  4. 会提示你 已经打开了开发者模式
  5. 进入到“系统和更新”，打开“开发人员选项”
  6. 打开“USB调试”【华为手机 会自动设置为OFF】

第三步：手机连接到电脑

第四步：打开命令行

  • Mac：打开 console 应用
  • Windows：搜索cmd，右键，管理员方式打开

第五步：开始删除

  1. 在命令行输入：

    abd devices

    如果你的手机已经连接成功并且开了开发者模式，会出现在反馈的结果里

  2. 常用的几个命令

    adb shell pm list packages #查看手机安装的所有软件包（包括不会出现在系统桌面上的app）

adb shell pm list packages | grep 'huawei' #搜索所有包含 huawei 关键字的包（单引号是 英文模式下的）

adb shell pm uninstall --user 0 com.baidu.input_huawei #这是卸载华为百度输入法的命令，com.baidu.input_huawei 是包名，（要删除其他软件，只要把包名更换即可）

附录：华为常见包的卸载方式

⚠️：如果出现删除失败 Failure [not installed for 0]，那就是你输入的包名在手机里没找到，华为的包名会变的，注意一下。（如果变了，就用上文搜索的方式找到相关包）

卸载命令 具体应用
adb shell pm uninstall --user 0 com.android.browser 浏览器
adb shell pm uninstall --user 0 com.android.cts.priv.ctsshim 旅行助手
adb shell pm uninstall --user 0 com.android.dreams.basic 基本互动屏保
adb shell pm uninstall --user 0 com.android.keyguard 华为杂志锁屏
adb shell pm uninstall --user 0 com.android.stk SIM卡应用
adb shell pm uninstall --user 0 com.baidu.input_huawei 百度输入法华为版
adb shell pm uninstall --user 0 com.huawei.android.hwpay 安全支付
adb shell pm uninstall --user 0 com.huawei.android.karaoke 卡拉ok
adb shell pm uninstall --user 0 com.huawei.appmarket 应用市场
adb shell pm uninstall --user 0 com.huawei.bd 用户体验计划
